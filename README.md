# Homebrew Tap

Homebrew formulae for [BrennerSpear](https://github.com/BrennerSpear) tools.

## Install

```bash
brew tap BrennerSpear/tap
brew install agent-tool-docs
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `agent-tool-docs` | Auto-generate agent-optimized CLI docs from --help output |
